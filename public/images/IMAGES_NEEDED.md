## Image Optimization

Before adding images to the website:

1. **Resize** images to recommended dimensions
2. **Optimize** using tools like:
   - TinyPNG (https://tinypng.com)
   - ImageOptim
   - Squoosh (https://squoosh.app)
3. **Convert** to WebP format for better performance (keep JPG as fallback)
4. **Compress** to keep file sizes under:
   - Hero/Large images: < 500KB
   - Gallery images: < 300KB
   - Logos: < 100KB
